# Hospital-Management-Dashboard
🏥 Hospital Emergency Room Dashboard

📌 1. Project Title / Headline

Hospital Emergency Room Analytics Dashboard
An interactive Power BI dashboard to analyze patient flow, wait time, admissions, and hospital performance.


📖 2. Short Description / Purpose

This dashboard provides a comprehensive analysis of hospital emergency room data. It helps healthcare professionals monitor patient trends, waiting time, satisfaction levels, and department referrals to improve operational efficiency and patient care.


🛠️ 3. Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 Power BI Desktop – Data visualization and dashboard creation
- 🔄 Power Query – Data cleaning and transformation
- 🧮 DAX (Data Analysis Expressions) – Calculated measures and KPIs
- 📁 Excel / CSV – Data source
- 🌐 GitHub – Project hosting and version control


📂 4. Data Source

- Source: Hospital Emergency Room Dataset
- Format: Excel / CSV
- Data Includes:
  - Patient Gender
  - Age Group
  - Admission Status
  - Department Referral
  - Wait Time
  - Satisfaction Score
  - Patient Race


📊 5. Features / Highlights

🔹 Business Problem

Hospitals need a clear and interactive system to monitor patient flow, reduce waiting time, and improve service quality in emergency rooms.

🎯 Goal of the Dashboard

- Analyze patient distribution and admission trends
- Monitor average waiting time
- Evaluate patient satisfaction
- Understand department workload

📈 Key Visuals (Charts Used)

- KPI Cards (Total Patients, Avg Wait Time, Avg Satisfaction)
- Bar Chart (Wait Time by Race & Gender)
- Horizontal Bar Chart (Department Referral, Age Group)
- Donut Chart (Patient Gender Distribution)
- Pie Chart (Patient Attended Within Time)
- Line / Area Chart (Satisfaction Score by Race)
- Slicers (Month, Gender, Admission Status)
